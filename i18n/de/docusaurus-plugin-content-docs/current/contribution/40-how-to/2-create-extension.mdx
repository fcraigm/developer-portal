# Wie reiche ich eine Extension ein?

Nachdem du dich [als Contributor registriert](../become-contributor)
hast kannst du eine neue Extension über unsere REST API registrieren.

<OperationLink operation="extension-register-extension" />

Sobald deine Extension fertig entwickelt und einsatzbereit ist,
kannst du über unsere REST API eine Überprüfung beantragen.

<OperationLink operation="extension-request-extension-verification" />

Wir werden den Inhalt deiner Extension prüfen und sie verifizieren.  
Nach erfolgreicher Verifizierung kannst du sie veröffentlichen.

<OperationLink operation="extension-set-extension-published-state" />

# Wie erstelle ich einen OAuth2-Client?

Wenn du für deine Extension einen OAuth2-Client nutzen möchtest, benötigen wir von dir nur ein ausgefülltes YAML-Dokument.
Dieses stellst du uns am Besten über ein Issue [in diesem privaten GitHub-Repository](https://github.com/mittwald/contributor-support) zur Verfügung.
Für den Aufbau und Inhalt der Datei, siehe [OAuth2 Referenz](../../reference/oauth2).

# Wie überarbeite ich meine Extension?

Der Code deiner Extension liegt in deiner Hand.
Wenn du Änderungen an der Konfiguration der Extension vornehmen möchtest ist dies über unsere REST API möglich.

<OperationLink operation="extension-patch-extension" />
