<APIMetadata version="v2" baseURL="https://api.mittwald.de/v2" />

---

Diese Referenz enthält eine Übersicht über die verfügbaren API-Endpunkte der mittwald mStudio v2 API. Diese Dokumentation wird aus der OpenAPI v3-Spezifikation generiert, die unter [https://api.mittwald.de/v2/openapi.json](https://api.mittwald.de/v2/openapi.json) zu finden ist.

Wirf einen Blick auf den [Umgang mit der API](/docs/v2/category/api)-Abschnitt dieser Dokumentation, um zu erfahren, wie du mit der API beginnen kannst.

## Änderungsprotokoll

Änderungen an der API werden im [Change Log](/changelog/tags/apiv2/) dokumentiert. Die Einträge im Änderungsprotokoll werden automatisch am Ende eines jeden Tages generiert. Wenn du über Änderungen an der API informiert werden möchtest, kannst du den [RSS-Feed](/changelog/rss.xml) abonnieren.
