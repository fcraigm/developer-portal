# Benutzerverwaltung

## Ohne eigene Nutzerverwaltung

Um die Entwicklung und Nutzung von Extensions zu vereinfachen, existiert die Möglichkeit, statt einer eigenen Nutzerverwaltung die existierenden mStudio Benutzer zu verwenden. Insbesondere, wenn die Extension dafür gedacht ist, hauptsächlich Domain Actions bei mittwald auszuführen, ist dies die einfachste Möglichkeit.

## Eigene Nutzerverwaltung

Extensions, die eine eigene Benutzerverwaltung erfordern, können sich das Extension Instance Secret zunutze machen, das über Lebenszyklus Ereignisse an die externe Anwendung übermittelt wurde. Mit diesem Secret können Access Tokens ohne Bezug auf einen mStudio Nutzer für die mStudio API ausgestellt werden. Diese Access Tokens können dann für Domain Actions verwendet werden, die entweder von den selbst verwalteten Benutzern oder autonom gestellt werden.
