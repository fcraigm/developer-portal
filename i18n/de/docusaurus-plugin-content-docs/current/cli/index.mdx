## Fehlerberichte, Feature-Anfragen und Feedback

Bitte benutze den [GitHub Issue Tracker](https://github.com/mittwald/cli/issues/new/choose), um Fehler zu melden und Features anzufragen. Für allgemeines Feedback oder Fragen kannst du gerne eine neue [GitHub-Diskussion](https://github.com/mittwald/cli/discussions) eröffnen. Bei sicherheitsrelevanten Problemen richte dich bitte nach unserer [Security Policy](https://github.com/mittwald/cli/security/policy).
