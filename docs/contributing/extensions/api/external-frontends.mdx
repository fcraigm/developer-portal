## URL-Templates

In den URLs, die den Usern die automatische Anmeldung in der Extension aus dem mStudio heraus ermöglichen, können Platzhalter verwendet werden.

Um das Single Sign On-Feature zu nutzen, sind folgende Parameter erforderlich:

- `accessTokenRetrievalKey`
- `userId`

Weitere mögliche Platzhalter sind:

- `apiVersion`
- `contributorId`
- `extensionId`
- `extensionInstanceId`

Die Platzhalter können folgendermaßen verwendet werden:

`https://example.com/sso/:apiVersion/extensions/:extensionId?userId=:userId&accessTokenRetrievalKey=:accessTokenRetrievalKey`
