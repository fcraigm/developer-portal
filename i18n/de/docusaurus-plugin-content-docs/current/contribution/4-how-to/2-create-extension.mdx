## Wie reiche ich eine Extension ein?

Um deine Extension für dich bereitstellen zu können, benötigen wir von dir nur ein ausgefülltes YAML-Dokument.
Dieses stellst du uns am besten über ein Issue [in diesem GitHub Repository](https://github.com/mittwald/contributor-support) zur Verfügung.

Wir werden einmal den Inhalt deiner Extension sichten und sie anschließend zur Installation für alle mStudio-Benutzer freischalten.
Für den Aufbau und Inhalt der Datei, siehe [Extension Referenz](../../reference/extensions).

## Wie erstelle ich einen OAuth2-Client?

Wenn du für deine Extension einen OAuth2-Client nutzen möchtest, benötigen wir von dir nur ein ausgefülltes YAML-Dokument. Dieses stellst du uns am Besten über ein Issue in GitHub zur Verfügung.
Für den Aufbau und Inhalt der Datei, siehe [OAuth2 Client Referenz](../../reference/oauth2).

## Wie überarbeite ich meine Extension?

Der Code deiner Extension liegt in deiner Hand.
Du kannst deinen Code ändern, wann und wie viel du magst.
Wenn du die Stammdaten deiner Extension ändern möchtest, schicke uns einfach das angepasste YAML-Dokument über ein GitHub-Issue mit einer kurzen Info, was sich geändert hat.
Wir werden die Anpassungen schnellstmöglich für dich durchführen.
